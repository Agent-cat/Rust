- We can declare function with `fn` keyword
```rust 
fn main (){
greet();
}
fn greet(){
    println!("Hi");
}
```
- We can pass argmunts and return the values from function
```rust
fn main (){
greet(a:i32,b:i32);
}
fn add(num1:i32 ,num2:132)->i32 //return type 
{
    num1+num2 //No need to type return if we dont add ; at end 
}
```
